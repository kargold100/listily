Listily Diagnostic Test

This page tests whether the site can load its data properly. If all checks pass below but the homepage still shows nothing, do a hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac).

Common fixes

  • Listings not showing on homepage? Hard refresh: Ctrl+Shift+R or Cmd+Shift+R
  • Still nothing? Open DevTools (F12) → Console tab. Look for red error messages and share them.
  • Backend not working? Go to Admin → Backend setup → Test connection. The diagnostic will tell you which step failed.
