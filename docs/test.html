System diagnostics

Common issues & fixes

  • "I see 'No pending submissions' on admin": That's correct — all 100 listings are pre-approved. Check the Approved tab to see them.
  • "Bulk upload not working": Make sure you've logged in to admin first (URL must be /admin.html, not the public site).
  • "Backend setup not working": First save the Sheets URL, the page will auto-reload. Then click "Test connection" — it'll tell you exactly what's wrong if anything.
  • "Changes not showing": Hard refresh — Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac) to bypass browser cache.
  • "Listings missing": Open DevTools (F12) → Console tab → look for red errors. Share them and I can fix.
