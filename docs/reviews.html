Want reviews visible to everyone?

Right now reviews are stored in each visitor's own browser — so they're not shared across devices. To make reviews public to all visitors, connect a free shared backend:

📊

Google Sheets

Easiest — uses Google Apps Script as a webhook. Free, no signup beyond a Google account.

🔥

Firebase

Google's realtime DB — free tier handles 100k reads/day. Already trusted by millions of apps.

⚡

Supabase

Open-source Postgres backend. 500MB and 50k monthly active users on the free tier.

See REVIEWS_BACKEND_SETUP.md in the repo for step-by-step setup for each option.
